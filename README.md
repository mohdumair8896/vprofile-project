## Deployment

To deploy this project run

```bash
  git clone https://github.com/mohdumair8896/vprofile-project.git
```

```bash
  cd vprofile-project\vagrant\Automated_provisioning_WinMacIntel\ 
```

```bash
  vagrant up
```
